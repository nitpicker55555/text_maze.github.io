# Text Maze 3D Visualization

This repository hosts an interactive 3D visualization of a text-based maze game.

## View the Visualization

Visit: https://nitpicker55555.github.io/text_maze.github.io/

Or view the specific visualization: https://nitpicker55555.github.io/text_maze.github.io/3d-graph-visualization.html

## Features

- Interactive 3D graph visualization
- Step-by-step playback of game progression
- Highlighting of new nodes and edges at each step
- Walkthrough display showing actions and observations
- Keyboard shortcuts (arrows for navigation, space for play/pause)

## Controls

- **Previous/Next buttons**: Navigate through steps
- **Play/Pause button**: Auto-play through all steps
- **Slider**: Jump to any step
- **Mouse**: Rotate, zoom, and pan the 3D view
- **Keyboard**: 
  - Left/Right arrows: Previous/Next step
  - Space: Play/Pause
